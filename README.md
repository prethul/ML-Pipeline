This project focuses on understanding and implementing a complete Machine Learning Pipeline using Python and Scikit-learn. It demonstrates how data preprocessing, feature transformation, and model training can be combined into a single streamlined workflow.

📚 What I Learned
What a Machine Learning Pipeline is and why it is used
How data flows through each stage of a pipeline
Working with ColumnTransformer for handling different feature types
How preprocessing and model training are combined efficiently
How a trained model makes predictions on new data
⚙️ Workflow
Data Collection
Data Preprocessing
Feature Transformation using ColumnTransformer
Model Training
Prediction on New Data
🎯 Key Concept

Instead of manually applying each preprocessing step, the pipeline automates the workflow and ensures consistency between training and testing phases.

Python
Scikit-learn
Pandas
NumPy

This project is part of my ML learning journey to understand how real-world machine learning systems are built in a structured and scalable way.
